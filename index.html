Hexo GitHub,搭建个人博客

准备工作

  1. 注册一个GitHub账号
  2. 本机安装Git
  3. 本机安装NodeJS

GitHub配置

  1. 新建仓库，仓库的名称为：你的账户名.github.io，例如：jianghang.github.io
  2. 进入仓库，在Settings页签下，往下拉至GitHub Pages，如下图：

Hexo配置

  1. 安装Hexo

    npm install hexo-cli -g

  2. 在当前目录下，初始化一个博客项目blog

    hexo init blog

  3. 进入blog目录

    cd blog

  4. 拉去依赖

    npm install

  5. 启动本地服务

    hexo server
  6. 访问http://localhost:4000

Hexo推送至GitHub仓库

  1. 安装插件

    npm install hexo-deployer-git --save

  2. 修改配置文件，进入blog文件夹，可以看到_config.yml配置文件，找到deploy配置项，如下：

    deploy:
  type: git
  repo: https://github.com/jianghang/jianghang.github.io.git
  branch: master

  3. 配置完毕，即可使用如下命令推送至目标仓库

    hexo deploy

Hexo默认主题在国内使用优化

添加favicon

  1. 修改favicon，默认主题没有favicon

    cd blog/themes/landscape/source
mkdir images

  2. 在images放入准备好的图标，例如：favicon.ico

  3. 修改_config.yml,在blog/themes/landscape目录下

    favicon: /images/favicon.ico
修改jquery插件CDN为国内的
  1. 默认的Hexo主题使用了jquery这个插件，使用的cdn是谷歌的
  2. 修改的文件路径为：blog/themes/landscape/layout/_partial/after-footer.ejs

  3. 找到如下代码：

    <script src="//ajax.googleapis.com/ajax/libs/jquery/2.0.3/jquery.min.js"></script>

  4. 修改为百度的CDN

    <script src="//apps.bdimg.com/libs/jquery/2.0.3/jquery.min.js"></script>

Hexo常用命令

  1. 生成静态文件

    hexo generate
hexo g

  2. 启动服务

    hexo server
hexo s

  3. 推送至仓库

    hexo deploy
hexo d

  4. 写作

    hexo new [layout] <title>
layout（布局）:
布局        路径
post        source/_posts
page        source
draft        source/_drafts
